## Como clonar este projeto

### 1. Clone o repositório

```
git clone https://github.com/mannoeu/MooveTV.git
```

### 2. Mude a pasta

```
cd MoovieTV
```

### 3. Instale as dependências

```
npm install
```

### 4. Gere a build

```
npm run build
```
